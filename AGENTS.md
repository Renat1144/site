# AGENTS.md

## Назначение проекта
Это статический сайт ООО «Современная упаковка» для презентации производства гофроупаковки в Пензе. Сайт продаёт расчёт и изготовление упаковки: гофрокороба, листовой гофрокартон, Z-картон, сложную высечку и вспомогательную упаковку.

Главное сообщение сайта из `index.html`: «Гофроупаковка, которая работает на ваш товар». Основной оффер: производство коробок, листового гофрокартона, Z-картона и сложной высечки под бизнес, перевозки и продажи.

## Источник правды
- Корневого `README` у основного сайта нет. Не выдумывать вводный текст, позиционирование, цифры и категории продукции: брать их из `index.html`, `script.js`, `styles.css` и фактических файлов проекта.
- `animated-bar-chart/README.md` относится только к отдельному Remotion-подпроекту, не к лендингу.
- Если нужен новый маркетинговый текст, сначала проверить существующий контент сайта. Если факта нет в проекте, явно отметить это и предложить формулировку как редакторскую, а не как подтверждённый факт.

## Ключевые файлы
- `index.html` — структура лендинга, SEO/meta, навигация, тексты, продуктовые блоки, контакты, ссылки на медиа.
- `styles.css` — вся визуальная система сайта: цвета, типографика, отступы, сетки, адаптивность, карточки и состояния.
- `script.js` — поведение сайта: мобильное меню, сброс прокрутки наверх при открытии, reveal-анимации, счётчики, витрина продукции, видео в разделе «О нас», форма заявки.
- `server.js` — простой Node.js static server, порт по умолчанию `8081`.
- `Logo.png` и `logo.svg` — фирменный знак. Для видимых логотипов сверяться с логотипом в шапке сайта.
- `архив/` — неиспользуемые медиа. Не подключать файлы из архива без явного решения вернуть их в активные ассеты.
- `animated-bar-chart/` — отдельный Remotion-проект с композициями `BarChart` и `FactoryLoop`; запуск через `npm run start`, рендер через `npm run render` или `npm run render:factory`.

## Визуальная система
Перед редизайном читать `styles.css`, а не подбирать стиль на глаз. Основные токены:
- тёмно-синий: `--navy: #11105a`, `--navy-2: #211f78`, `--navy-3: #303099`;
- оранжевый: `--orange: #f05a28`, `--orange-2: #ff7846`;
- фон: `--paper: #f5f7fb`, `--white: #ffffff`;
- текст второго уровня: `--muted: #647086`;
- радиус: `--radius: 14px`;
- максимальная ширина: `--max: 1680px`;
- базовые интервалы: `--section-y`, `--section-y-tight`, `--subsection-gap`.

Шрифт сайта задан как `Inter, "Segoe UI", Arial, sans-serif`. Крупные заголовки уже были уменьшены для более спокойного вида; не возвращать чрезмерно большие плотные заглавные блоки без отдельного запроса.

## Правила дизайна для этого сайта
- Сначала читать существующую страницу и код, потом проектировать. Структура сайта уже задана как производственный каталог с витриной продукции, услугами, производством, компанией, инфоцентром, соцсетями и контактами.
- Не добавлять placeholder-контент, абстрактные лозунги, случайные иконки и новые разделы без опоры на существующий проект или прямое ТЗ.
- Сохранять фирменную палитру: тёмно-синий, оранжевый, белый. Градиенты ранее убирались по запросу пользователя; не возвращать их без явного согласования.
- Следить за едиными вертикальными отступами между разделами. На сайте уже есть токены интервалов, лучше менять их централизованно.
- Для медиа проверять реальные имена файлов и расширения: в проекте используются `.jpg`, `.png`, `.webm`, `.mp4`, `.svg`.
- При замене изображения на видео использовать `autoplay muted loop playsinline`, если видео должно работать как бесконечная анимация без плеера.

## Поведение и интерактивность
- Витрина продукции управляется объектом `products` в `script.js` и кнопками `data-product-tab` в `index.html`.
- Слайдер продукции запускается через `startProductSlider()`; скорость меняется в интервале `setInterval`.
- Видео раздела «О нас» управляется атрибутами `data-about-video` и `data-about-video-play`.
- При открытии страницы скрипт сбрасывает прокрутку наверх через `history.scrollRestoration = "manual"` и `window.scrollTo(0, 0)`.
- Форма заявки сейчас не отправляет данные на сервер: она показывает сообщение-заглушку о необходимости подключить обработчик.
